# 💬 Chatbot con Historial usando la API de Gemini + ipywidgets

Este proyecto implementa un **chatbot conversacional con memoria de historial** utilizando la API de Google Gemini. La interfaz interactiva está construida con **ipywidgets** directamente dentro del notebook, permitiendo al usuario chatear con la IA sin salir del entorno de Google Colab. El modelo recuerda el contexto de toda la conversación en tiempo real.

---

## 📌 Características

- Interfaz de chat interactiva dentro del notebook (sin salir de Colab).
- Historial de conversación persistente durante toda la sesión.
- Botón para iniciar una nueva conversación y limpiar el historial.
- Compatible con el modelo `gemini-1.5-flash` (rápido y gratuito).
- Manejo de errores de API con mensajes claros al usuario.

## 🔑 Requisito previo: API Key de Gemini

Este proyecto requiere una **API Key gratuita de Google AI Studio**.

1. Ve a 👉 [https://aistudio.google.com](https://aistudio.google.com)
2. Haz clic en **"Get API Key"**
3. Copia tu clave y pégala en la celda de configuración del notebook
4. *(Opcional)* Usa los **Secrets** de Colab (ícono 🔑 en el panel izquierdo) para no exponer tu clave

> ⚠️ **Nunca compartas ni subas tu API Key a GitHub.**
